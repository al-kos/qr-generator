Лабораторная работа по дисциплине:

"Основы web технологий"

Краткое описание проекта:

При нажатии на кнопку Generate QR-Code сгенерируется картинка с кодом, ведущая на страницу с введенным пользователем URL-адресом. По умолчанию указан адрес сервиса для генерации кода.

Целью данного проекта является освоение языка разметки HTML5, таблицам стилей CSS3 и основ языка программирования Javascript для пользовательского взаимодействия.

Задачи:

  1. Использование html-тегов и атрибутов для работы с текстом;
  2. Стилизация html-документа с использованием каскадных таблиц стилей (css);
  3. Использование скрипов на языке JavaScript.

Группа ИСТ-2022, выполнил Косенко А.С.
