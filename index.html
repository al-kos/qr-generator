Краткое описание проекта:

При нажатии на кнопку Generate QR-Code сгенерируется картинка с кодом, ведущая на страницу с введенным пользователем URL-адресом. По умолчанию указан адрес сервиса для генерации кода

Целью данного проекта является создание QR-кода по введенному пользователем URL-адресу.

Задачи и их реализация

  1. Использование html-тегов и атрибутов для работы с текстом:
    • В проекте используются теги блоков, заголовков, абзацев, пользовательского ввода, кнопки, списки
    • Большинству тегов определены атрибуты class или id для дальнейшей стилизации и обращению в скрипте
    • Для тега кнопки используются атрибут события по нажатию кнопки
  2. Стилизация html-документа с использованием каскадных таблиц стилей (css):
    • В проекте используются css-переменные для удобства работы с цветами
    • Стили присвоены с использованием различных атрибутов (class, id)
    • Для тега кнопки используются атрибут события по нажатию кнопки
  3. Использование скрипов на языке JavaScript:
    • Получение доступа к элементам html-документа по их атрибуту id
    • Получение и отображение изображения по нажатию кнопки мыши
